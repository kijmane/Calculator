## Lv.3에서 추가된 기능

계산 히스토리 조회 기능 추가
- 사용자가 계산한 결과를 저장하고 계산이 끝날 때마다 마지막 계산 결과와 전체 계산 히스토리를 출력하는 기능을 구현했습니다.
- 히스토리에는 사용자가 입력한 연산 결과들이 순차적으로 저장됩니다.
- EX ) 첫번째 계산 결과: 30.0
계산 히스토리: [30.0, 25.0]
첫번째 숫자를 입력해주세요. : 1
두번째 숫자를 입력해주세요. : 2
연산 기호를 입력해주세요. (+,-,*,/) : +
계산 결과 :  3.0
마지막 계산 결과 : 3.0
계산 히스토리: 
30.0
25.0
3.0
더 계산하시겠습니까? (exit 입력 시 종료됩니다.) :
